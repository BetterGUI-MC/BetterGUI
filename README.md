## Feature
* Modular design (Base plugin with addons)
* Simple settings
* Animated Icons
* Priority in icons
* Special commands
* View & Click Requirements
* Cooldown
## Building
1. Clone this repo
2. Open the folder in terminal
3. Type `mvn clean install`
4. Get the final file in `/target/`
